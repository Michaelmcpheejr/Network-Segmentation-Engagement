# Scope

## Objective
Assess the effectiveness of network segmentation controls and identify risks associated with lateral movement and improper access boundaries.

## In Scope
- VLAN configuration and segmentation design
- Trunk link configurations
- Firewall zones and policies
- Router-level access control mechanisms
- Internal traffic visibility and monitoring

## Out of Scope
- Endpoint security
- Application-layer vulnerabilities
- External threat simulation

## Assessment Approach
- Configuration review (switches, routers, firewall)
- Interview with network engineer
- Risk analysis and scoring
- Framework mapping (NIST 800-53, NIST CSF, CIS Controls, MITRE ATT&CK)
